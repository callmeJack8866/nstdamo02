QUOTE DRAFT · 卖方报价草稿

卖方非标报价预览

Struct Ready

在左侧输入条款后，这里会生成一段「人类可读」+「合约可解析」的卖方报价摘要，展示给买方与平台风控。

合约核心字段示例（链上 quote struct）

struct Quote {
    uint256  orderId;              // 对应买方订单（可选）
    address  seller;
    string   underlyingName;
    string   market;
    bytes32  direction;            // CALL / PUT / BOTH
    uint256  notionalMin;
    uint256  notionalMax;
    bytes32  notionalCurrency;

    uint32   tenorMin;             // 单位：天
    uint32   tenorMax;             // 单位：天
    uint16   premiumMinBps;        // 年化下限：1000 = 10.00%
    uint16   premiumMaxBps;        // 年化上限：1800 = 18.00%

    uint16   marginRateBps;        // 保证金率：3000 = 30.00%
    bytes32  marginAsset;
    bytes32  marginMode;
    bytes32  hedgeMode;

    uint16   maxOrders;
    uint256  maxExposurePerBuyer;
    bytes32  sellerRole;
    bool     allowJointSellers;

    string   hedgeNote;
    string   riskNote;
}

如何「一键上 DApp」？

  • 把上面的表单值映射成 Quote 结构体参数。
  • 前端通过钱包（如 MetaMask / OKX）调用 submitQuote(quote) 或 quoteFor(orderId, quote)。
  • 成功后返回 quoteId，可在订单大厅展示卖方报价列表。
